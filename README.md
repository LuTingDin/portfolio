# Lue's Portfolio
 
個人作品集網站，展示前端開發專案與技術能力。
 
🔗 **Live Demo:** https://lutingdin.github.io/portfolio/
 
---
 
## 技術棧
 
- HTML5 / CSS3 / Vanilla JavaScript
- 響應式設計
- CSS 自定義變數系統
- GitHub Pages 部署
---
 
## 專案展示
 
| 專案 | 說明 | Demo |
|------|------|------|
| 🛒 購物網站 | 多頁面電商平台，購物車、localStorage 狀態管理 | [Live Demo](https://lutingdin.github.io/shopping-site/) |
| 🌦️ 天氣 App | 串接 OpenWeatherMap API，Promise.all 並行請求 | [Live Demo](https://lutingdin.github.io/weather-app/) |
| 💰 記帳 App | 完整 CRUD、Chart.js 圖表視覺化、多重篩選 | [Live Demo](https://lutingdin.github.io/expense-tracker/) |
